git常见的命令工具

#git的常见命令操作

删除文件的方法

git rm 文件名  （多个可以加空格后加上）

恢复最近提交的一个状态

git checkout HEAD -- 文件名

查看本地提交的架构ID

git log --oneline

恢复ID提交

git revent (要恢复的ID)

#控制指针的提交

恢复到指定的提交文件，其他的暂存，如果提交远程会覆盖

git reset --soft ID

恢复到你指定的指针，其他的ID也是看的见得

git reset --hard ID

#控制分支

控制分支的方法

git branch "分支的名字"

查看所有的分支

git branch

切换分支

git checkout "分支的名"

查看ID信息与提交的分支

git log --oneline --decorate

比较两个分支之间的区别

git diff "分支名".."分支名"

合并分支，前提要合并的分支都提交完成了

git merge "要合并的分支名"

合并如果同时修改同一个文件的时候就会出现冲突

要解决后再合并分支

重命名分支

git branch -m "分支名" "修改的分支名"

删除分支

git branch -d "分支名"

#远程的版本库

创建远程的版本库

git remote add origin "远程仓库的地址";

查看远程的仓库地址

git remote -v

远程推送文件

git push -u origin "分支名"

同步远程更新的分支获取到本地

git fetch

远程分支获取状态更新

git merge origin/"分支名"
